Paste the Web App URL from your deployed Google Apps Script. Every scan will POST to this URL and write a new row into your Google Sheet automatically.
Apps Script Code
Deploy this at script.google.com → New project → Deploy as Web App
function doPost(e) { var sheetId = 'YOUR_SHEET_ID_HERE'; // from Sheet URL var ss = SpreadsheetApp.openById(sheetId); var sheet = ss.getSheetByName('Scans') || ss.getActiveSheet(); var data = JSON.parse(e.postData.contents); if (sheet.getLastRow() === 0) { sheet.appendRow(['ScanID','Model','RawData','Date','Time','Status']); } sheet.appendRow([ data.scanId, data.model, data.rawData, data.date, data.time, data.status || 'OK' ]); return ContentService .createTextResponse(JSON.stringify({result:'OK'})) .setMimeType(ContentService.MimeType.JSON); }
Replace YOUR_SHEET_ID_HERE with the ID in your Sheet URL. Deploy → Execute as: Me → Who has access: Anyone.
